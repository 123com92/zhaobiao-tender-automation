# 用户操作手册

## 1. 启动程序

双击运行：

```text
tender_automation_gui.exe
```

如果桌面存在快捷方式，可以直接打开“招标信息自动汇报工具”。

## 2. 设置关键词

在“关键词”输入框中填写需要关注的词，例如：

```text
空调、维保、废品回收
```

支持以下分隔方式：

- 顿号：`空调、维保`
- 逗号：`空调,维保`
- 分号：`空调;维保`
- 换行：每行一个关键词

填写后点击“保存关键词”。保存内容会写入：

```text
D:\TenderDocs\keywords.txt
```

## 3. 设置爬取网址

在网址输入框中填写招标列表页地址，然后点击“添加网址”。

当前已重点适配江苏省公共资源交易相关页面，例如：

```text
http://jsggzy.jszwfw.gov.cn/
```

通用网页也会尝试解析页面链接和标题，但不同网站结构差异较大，抓取效果可能不同。

## 4. 生成报告

点击“生成报告”，程序会执行：

1. 读取网址配置。
2. 读取关键词配置。
3. 抓取招标信息。
4. 过滤非招标/采购阶段的信息。
5. 按历史记录做增量去重。
6. 生成 Excel 简报。

报告位置：

```text
D:\TenderDocs\最新招标信息汇报.xlsx
```

## 5. 发送到微信

发送前确认：

- 电脑版微信已登录。
- 微信主窗口可以正常显示。
- 不要在发送过程中操作鼠标键盘。

点击“生成报告并发送微信”，程序会：

1. 生成最新 Excel 报告。
2. 查找微信主窗口。
3. 搜索“文件传输助手”。
4. 将 Excel 文件复制到剪贴板。
5. 粘贴并发送文件。

## 6. 定时自动执行

勾选“启用定时自动发送”，设置上午和下午两个时间，例如：

```text
08:00
14:00
```

点击“保存定时”。

注意：

- 当前定时功能由 GUI 程序内部计时器实现。
- 程序必须保持运行。
- 电脑需要处于可操作桌面环境。
- 微信需要保持登录。
- 如果电脑锁屏、休眠或微信弹窗异常，微信自动发送可能失败。

## 7. 清空处理历史

如果想重新汇总之前已经汇报过的项目，点击“清空处理历史”。

这会清空：

```text
D:\TenderDocs\history_log.txt
```

清空后，同一批招标信息可能重新出现在报告中。

## 8. 常见问题

### 报告没有信息

可能原因：

- 关键词过窄。
- 网站当天没有匹配项目。
- 增量历史已经记录过这些项目。
- 当前网站结构变化，导致解析不到标题或详情。

处理方式：

- 增加关键词。
- 点击“清空处理历史”后再生成一次。
- 检查网址是否能在浏览器中正常打开。

### Excel 打不开

新版已经改为真正的 `.xlsx` 文件，并做了 UTF-8 清洗。如果仍打不开，先确认打开的是：

```text
D:\TenderDocs\最新招标信息汇报.xlsx
```

不要打开旧的 `.xls` 文件。

### 微信发送失败

可能原因：

- 微信没有登录。
- 微信窗口被最小化或处于异常弹窗。
- 电脑锁屏。
- 文件传输助手搜索结果没有进入聊天。

处理方式：

- 手动打开微信主窗口。
- 确认能看到文件传输助手。
- 重新点击“生成报告并发送微信”。
